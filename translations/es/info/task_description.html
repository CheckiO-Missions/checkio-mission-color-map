En matemáticas el teorema de los cuatro colores, (o el teorema de mapa de cuatro colores) establece que: cuando se da alguna separación de un plano en regiones contiguas y se produce una figura llamada mapa, no se requieren más de cuatro colores para colorear las regiones del mapa, de manera que hayan dos regiones adyacentes con el mismo color. Dos regiones se llaman adyacentes si comparten un límite común que no sea una esquina, donde esquina hace referencia a puntos compartidos por tres o más regiones. Para nuestro modelo vamos a utilizar una cuadrícula con celdas cuadradas.

Se te asigna un mapa de regiones como una cuadrícula (matriz). Existen N países ubicados en este mapa, y cada país tiene un número de 0 a N-1. Dos celdas son adyacentes si tienen un borde común y cada país tiene una o más celdas que están conectadas. Te puedes mover entre las celdas del país X simplemente utilizando sus las celdas adyacentes.
Cada celda está marcada por el número de su país designado.

Deberás "colorear" un mapa usando 4 colores, teniendo en cuenta que todas las celdas que comprenden un país deberán tener un solo color, y las celdas adyacentes de diferentes países deben tener un color distinto.

El resultado debe ser representado como una secuencia de números 1,2,3 o 4, donde cada elemento muestra el color del país que coincide con el índice. Por ejemplo, el elemento 0th del resultado muestra el color del país 0. De esta forma, el resultado deberá tener N elementos.

Datos de Entrada: Un mapa de regiones, como una tupla de tuplas con enteros (int).

Salida: La secuencia de colores como una tupla / lista de enteros (int).

Ejemplo: 

color_map(((0, 0, 0), (0, 1, 1), (0, 0, 2))) # [1, 2, 3] or [2, 3, 1] or ...

¿Cómo se utiliza?: Este es un clásico problema de satisfacción de restricciones (CSP por sus siglas en inglés). El teorema de los cuatro colores fue probado en 1976 por Kenneth Appel y Wolfgang Haken. Este fue el primer teorema de importancia a ser probado usando un ordenador. El enfoque de Appel y Haken comenzó demostrando que hay un conjunto particular de 1.936 mapas, cada uno de los cuales no pueden ser parte de un contraejemplo de menor tamaño para el teorema de cuatro colores.

Condiciones: 0 < len(region) ≤ 10
all(0 < len(row) ≤ 10 and len(row) == len(region[0]) for row in region)
Las celdas pertenecientes a un país están conectadas.
Los números de los paises son secuencias de 0 hasta N-1.
