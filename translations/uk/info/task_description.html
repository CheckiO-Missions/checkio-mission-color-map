У математиці the four color theorem, або теорема про карту чотирьох кольорів, стверджує, що: при будь-якому розбитті площини на суміжні області на суміжні області і отримуємо фігуру, яка називається мапою, не більше чотирьох кольорів потрібно для того, щоб розфарбувати областей карти так, щоб жодні дві сусідні області не мали однакового кольору. Дві області називаються суміжними, якщо вони мають спільну межу, яка не є кутом, де кути - це точки спільні для трьох або більше областей. Для нашої моделі ми будемо використовувати сітку з квадратними комірками.

Вам задано карту регіону у вигляді сітки (матриці). На цій карті розташовано N країн. Кожна країна має номер від 0 до N-1. Дві клітинки вважаються суміжними, якщо вони мають спільний край. Кожна країна має одну або декілька клітинок, які з'єднані між собою. Таким чином, ви можете переміщатись між будь-якими клітинками країни X, просто використовуючи для цього сусідні клітинки.
Кожна клітинка позначена номером відповідної країни.

Ви повинні "розфарбувати" карту у 4 кольори. Всі клітинки, що складаються з однієї країни повинні бути одного кольору. Сусідні клітинки різних країн не повинні мати однаковий колір.

Результат повинен бути представлений у вигляді послідовності чисел 1,2,3 або 4. Кожен елемент показує колір країни, що відповідає індексу. Наприклад, 0-й елемент показує колір країни 0. Таким чином, результат повинен містити N елементів.

Вхідні дані: Карта області у вигляді кортежу кортежів з цілими числами.

Вихідні дані: Послідовність кольорів у вигляді кортежу/списку цілих чисел.

Приклад: 

color_map(((0, 0, 0), (0, 1, 1), (0, 0, 2))) # [1, 2, 3] or [2, 3, 1] or ...

Як використовується: Це класичний constraint satisfaction problem. Теорема про чотири кольори була доведена в 1976 році Кеннетом Аппелем і Вольфгангом Хакеном. Це була перша велика теорема, яку було доведено за допомогою комп'ютера. Підхід Аппеля та Хакена розпочався з того, що вони показали, що існує певний набір з 1,936 карт. Кожна з них не може належати до жодної з чотирьох кольорів, кожна з яких не може бути частиною найменшого за розміром контрприкладу до теореми про чотири кольори.

Умова: 0 < len(region) ≤ 10
all(0 < len(row) ≤ 10 і len(row) == len(region[0]) для рядка в регіоні)
З'єднано комірки з однією країною.
Номери країн - послідовність від 0 до N-1.
